## Create App Icons (Optional)

For App Store deployment, create icons:

1. Create a 512x512 PNG image
2. Save as `static/icons/icon-base.png`
3. Install Pillow: `pip install Pillow`
4. Run: `python create_icons.py`

Or use online tools listed in `static/icons/README.md`

## Troubleshooting

**App is slow?**
- Clear browser cache
- Check internet connection
- Restart the app

**Icons not showing?**
- Create icons using `create_icons.py`
- Or use placeholder icons from online tools

**Can't install as PWA?**
- Use Chrome/Edge browser
- Make sure you're on HTTPS (or localhost)
- Check that manifest.json is accessible
